## 🚀 Typical Developer Setup

```bash
# Update system
sudo apt update && sudo apt upgrade -y

# Install Git
sudo apt install git -y

# Install Node.js (example)
sudo apt install nodejs npm -y

# Install Docker
sudo apt install docker.io -y

# Verify installations
git --version
node -v
docker --version
```
